# Berechtigungen für Adressen

Hier können Adressen ausgewählt werden, die die Gruppe bearbeiten darf. Teilbaum bedeutet, dass der gesamt Teilbaum unterhalb der ausgewählten Adresse bearbeitet werden kann.
