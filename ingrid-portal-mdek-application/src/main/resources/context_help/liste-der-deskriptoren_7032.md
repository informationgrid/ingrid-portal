# Liste der Deskriptoren

Anzeige der aus dem Hierarchiebaum oder der Such-Ergebnisliste hinzugefügten Deskriptoren. Über den Button "Übernehmen" werden diese bei den Thesaurus-Suchbegriffen ergänzt.
