# Berechtigungen für Objekte

Hier können Objekte ausgewählt werden, die die Gruppe bearbeiten darf. Teilbaum bedeutet, dass der gesamt Teilbaum unterhalb des ausgewählten Objekts bearbeitet werden kann.
