# Zugriffsbeschränkungen

Das Feld „Zugriffsbeschränkungen“ beschreibt, die Art der Zugriffsbeschränkung. Bei frei nutzbaren Daten bzw. Services soll der Eintrag „keine“ ausgewählt werden (ISO: accessConstraints).
