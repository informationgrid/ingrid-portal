Desktop GIS:

In der WSV wird als Standard-GIS die Software ArcGIS der Firma ESRI eingesetzt.

ArcGIS ist eine Produktfamilie und setzt sich aus vielen den speziellen Aufgaben angepassten Client- und Serverlösungen zusammen.

Als Desktop-GIS stehen in der WSV eine Anzahl von Lizenzen von:

  • ArcGIS ArcView (universelles Desktop-GIS für allgemeine Anwendungen)
  • ArcGIS ArcInfo (erweitertes Desktop-GIS mit zusätzlichen, speziellen Funktionen)
  • Extension 3D Analyst (Aufbau, Visualisierung und Analyse von Digitalen Höhenmodellen)
  • Extension Spatial Analyst (Erweiterung für weitergehendeAnalysen im Rasterbereich)
  • Extension Publisher (Erweiterung zur Weitergabe von Kartenprojekten)

zur Verfügung.

ArcGIS via Citrix:

Während ArcGIS Desktop durch einen Administrator auf Ihrem PC installiert werden muss, kommt ArcGIS via Citrix ohne Installation aus und kann direkt aus dem Internet Explorer aufgerufen werden.

ArcMap / ArcCatalog laufen auf einem leistungsstarken Serververbund, der sog. Citrix-Zentralfarm, beim DLZ-IT. Sie arbeiten jedoch, als liefen die Anwendungen direkt auf Ihrem PC. Sie können lokale und entfernte Daten einladen und auf Ihre lokalen Drucker drucken.

Um diese Technologie nutzen zu können, müssen Sie sich einerseits freischalten lassen (Infos siehe rechts) und es muss einmalig ein sogenannter Citrix-Client auf ihrem PC installiert werden.
